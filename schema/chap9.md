# 제9장: GraphRAG/LightRAG/KAG — 관계 기반 추론으로 확장 — 집필 계획서

## 장 개요

이 장은 벡터 RAG의 한계를 극복하는 그래프 기반 RAG 기술을 다룬다. 학습자는 (1) 벡터 RAG의 한계, (2) 지식 그래프 개념, (3) GraphRAG와 LightRAG의 차이, (4) 실무 적용 시 고려사항을 익힌다. 실습에서는 동일 질의를 벡터 RAG와 그래프 기반 방식으로 비교한다.

### 학습 목표

1) 벡터 RAG의 한계와 그래프 기반 접근의 필요성을 이해한다.
2) 지식 그래프의 기본 개념(엔티티, 관계, 트리플)을 학습한다.
3) GraphRAG와 LightRAG의 아키텍처와 차이점을 비교한다.
4) 동일 질의로 벡터 RAG와 그래프 RAG의 결과를 비교 분석한다.
5) 그래프 기반 RAG의 적합한 사용 사례를 식별한다.

---

## 9.1 벡터 RAG의 한계

### 집필 방향

8장에서 구현한 벡터 RAG의 한계를 구체적인 예시와 함께 설명한다. 복잡한 관계 추론, 다중 홉 질문에서의 약점을 보여준다.

### 강화 항목(필수)

- 단순 유사도 매칭의 한계
- 다중 홉(multi-hop) 질문 처리 어려움
- 엔티티 간 관계 파악 불가
- 전역적 맥락 이해 부족

---

## 9.2 지식 그래프 기초

### 집필 방향

지식 그래프의 기본 개념을 소개한다. 엔티티, 관계, 트리플 구조를 설명하고, LLM을 활용한 지식 그래프 구축 방법을 다룬다.

### 강화 항목(필수)

- 엔티티(Entity)와 관계(Relation)
- 트리플(Triple): 주어-술어-목적어
- LLM 기반 엔티티/관계 추출
- 그래프 데이터베이스 개요 (Neo4j 등)

---

## 9.3 GraphRAG: Microsoft의 접근

### 집필 방향

Microsoft에서 개발한 GraphRAG의 아키텍처와 특징을 설명한다. 계층적 커뮤니티 요약, 글로벌/로컬 검색 전략을 다룬다.

### 강화 항목(필수)

- GraphRAG 아키텍처 개요
- 엔티티/관계 추출 파이프라인
- 커뮤니티 탐지와 계층적 요약
- 글로벌 검색 vs 로컬 검색
- 비용과 인덱싱 시간 고려

---

## 9.4 LightRAG: 경량화된 그래프 RAG

### 집필 방향

LightRAG의 설계 철학과 GraphRAG 대비 장점을 설명한다. 간소화된 파이프라인과 효율성을 강조한다.

### 강화 항목(필수)

- LightRAG 아키텍처
- GraphRAG와의 주요 차이점
- 듀얼 레벨 검색 (로컬 + 글로벌)
- 증분 업데이트 지원
- 성능과 비용 트레이드오프

### 표(권장)

**표 9.1** GraphRAG vs LightRAG 비교

| 기준 | GraphRAG | LightRAG |
|-----|----------|----------|
| 개발사 | Microsoft | HKUDS |
| 인덱싱 비용 | 높음 | 낮음 |
| 증분 업데이트 | 제한적 | 지원 |
| 검색 전략 | 글로벌/로컬 분리 | 듀얼 레벨 통합 |
| 적합한 사용 | 대규모 분석 | 빠른 프로토타입 |

### 강화 항목(필수)

- 도메인 특화 지식 그래프

## 9.6 실습: 벡터 RAG vs 그래프 RAG 비교

### 실습 목표

1) 동일한 문서 코퍼스를 사용하여 벡터 RAG와 LightRAG를 구현한다.
2) 동일한 질문 세트로 두 방식의 결과를 비교한다.
3) 각 방식의 장단점을 실제 결과로 분석한다.

### 작업 정의

Python asyncio 문서를 사용하여:
- 벡터 RAG (8장 코드 재사용)
- 그래프 기반 RAG (엔티티/관계 추출)
- 동일 질문 세트로 비교

### 필요 라이브러리

- `langchain-openai`: OpenAI LLM
- `chromadb`: 벡터 저장소
- `networkx`: 그래프 처리
- `python-dotenv`: 환경 변수

### 실습 코드

- `practice/chapter9/code/9-6-graph-rag.py`: 그래프 기반 RAG 구현
- `practice/chapter9/code/9-6-compare.py`: 벡터 vs 그래프 비교
- `practice/chapter9/code/requirements.txt`: 의존성 목록

### 실습 산출물(필수)

- `practice/chapter9/data/output/ch09_vector_result.json`: 벡터 RAG 결과
- `practice/chapter9/data/output/ch09_graph_result.json`: 그래프 RAG 결과
- `practice/chapter9/data/output/ch09_comparison.json`: 비교 분석
- `practice/chapter9/data/output/ch09_knowledge_graph.json`: 추출된 지식 그래프
