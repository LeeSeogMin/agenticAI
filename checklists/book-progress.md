## 7단계 워크플로우 To-Do (장 단위)

아래 To-Do는 각 장(`docs/ch{N}.md`)을 완성할 때 항상 동일하게 적용한다. 각 단계의 “완료”는 산출물이 파일로 존재하고, 필요한 경우 실제 실행으로 검증되었음을 의미한다.

| 단계 | 체크 | 산출물(예시) |
|---|---|---|
| 1. Planning | [ ] | `schema/chap{N}.md` |
| 2. Information Gathering | [ ] | `content/research/ch{N}-refs.md` |
| 3. Analysis | [ ] | (선택) `content/drafts/ch{N}-outline.md` |
| 4. Implementation & Documentation | [ ] | `practice/chapter{N}/code/`, `content/drafts/ch{N}.md` |
| 5. Optimization | [ ] | (선택) 리팩토링/구조 정리 |
| 6. Quality Verification | [ ] | `docs/ch{N}.md` 검토 반영 |
| 7. MS Word Conversion | [ ] | `ms-word/output/ch{N:02}.docx` (예: `ch01.docx`) |
