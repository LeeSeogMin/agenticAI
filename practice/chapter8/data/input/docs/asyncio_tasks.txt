# Python asyncio 태스크와 동시성

## Task 생성

Task는 코루틴을 감싸서 이벤트 루프에서 실행되도록 예약하는 객체다. asyncio.create_task() 함수를 사용하여 태스크를 생성한다.

```python
async def fetch_data(url):
    await asyncio.sleep(1)  # 네트워크 요청 시뮬레이션
    return f"Data from {url}"

async def main():
    task = asyncio.create_task(fetch_data("https://api.example.com"))
    # 다른 작업 수행 가능
    result = await task
    print(result)
```

태스크를 생성하면 코루틴이 즉시 스케줄링된다. await를 호출하면 태스크가 완료될 때까지 기다린다.

## asyncio.gather()

여러 코루틴을 동시에 실행하려면 asyncio.gather()를 사용한다. 모든 코루틴이 완료될 때까지 기다리고 결과를 리스트로 반환한다.

```python
async def main():
    results = await asyncio.gather(
        fetch_data("url1"),
        fetch_data("url2"),
        fetch_data("url3")
    )
    print(results)
```

gather()는 순서를 보장한다. 결과 리스트의 순서는 입력 코루틴의 순서와 동일하다.

## asyncio.wait()

더 세밀한 제어가 필요할 때는 asyncio.wait()를 사용한다. 완료된 태스크와 대기 중인 태스크를 구분하여 반환한다.

```python
async def main():
    tasks = [
        asyncio.create_task(fetch_data(f"url{i}"))
        for i in range(5)
    ]
    done, pending = await asyncio.wait(
        tasks,
        return_when=asyncio.FIRST_COMPLETED
    )
```

return_when 매개변수로 FIRST_COMPLETED, FIRST_EXCEPTION, ALL_COMPLETED를 지정할 수 있다.

## 타임아웃 처리

asyncio.wait_for()를 사용하여 코루틴에 타임아웃을 설정할 수 있다. 지정된 시간 내에 완료되지 않으면 asyncio.TimeoutError가 발생한다.

```python
async def main():
    try:
        result = await asyncio.wait_for(
            slow_operation(),
            timeout=5.0
        )
    except asyncio.TimeoutError:
        print("작업이 타임아웃되었습니다")
```

타임아웃이 발생하면 해당 태스크는 취소된다.
