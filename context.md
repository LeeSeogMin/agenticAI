# 프로젝트 목표: AI 에이전트 개발 교재 집필

이 프로젝트는 **AI 에이전트 개발: MCP·프레임워크·RAG·배포까지**라는 주제의 대중 교재를 6개 전문 에이전트(@planner, @researcher, @writer, @coder, @reviewer, @graphic)가 협력하여 집필·검증·변환하는 워크플로우 시스템이다.

- **대상 독자**: 기본 프로그래밍 경험이 있는 학부 고학년/대학원생, 현업 주니어 개발자
- **구성**: 12장 5파트 (AI 협업 → MCP/A2A → 에이전트 프레임워크 → RAG/지식 → 신뢰성/배포)
- **기준 시점**: 2026년 상반기. MCP 2025-11-25 스펙, A2A v0.3, LangGraph 1.0, OpenAI Agents SDK, Google ADK 기준
- **최종 목차**: `contents.md` 참조
- **집필 철학**: "왜 필요한가 → 어떻게 동작하는가" 순서, 코드보다 설계 판단 우선
